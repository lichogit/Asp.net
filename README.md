# üëü Cloud Strider - ASP.NET Core Sneaker Shop

Cloud Strider is a full-stack e-commerce web application built with **ASP.NET Core MVC** and **Entity Framework Core**. It features a modern, brutalist UI, a fully functional shopping cart, and a secure, role-based authentication system with OAuth integrations.

## ‚ú® Key Features

* **Modern UI/UX**: A clean, responsive, and minimalist design built with Bootstrap 5.
* **Custom Authentication**: Full implementation of ASP.NET Core Identity with custom registration and login forms.
* **OAuth Integration**: Users can seamlessly sign in or register using their **Google** accounts.
* **Role-Based Admin Dashboard**: A secure, hidden control panel restricted to "Admin" users, allowing them to manage the store's inventory.
* **Full CRUD Operations**: Admins can Create, Read, Update, and Delete sneakers directly from the web interface.
* **Dynamic Catalog & Search**: Users can browse the sneaker collection and use the search bar to filter shoes by name.
* **Session-Based Shopping Cart**: Users can add items to their cart, which is managed via temporary browser session state.
* **Automated Database Seeding**: The application automatically seeds initial sneaker data and configures the default Admin user on startup.
